
# 별 찍기 - 18 (10993)

https://www.acmicpc.net/problem/10993

## Approach

당연히, 홀짝 구분해야함

접근 순서는 :

1. 테두리 + 중간 => 별
2. 안쪽 삼각형 => 재귀
3. N=3일때 마지막(예외 아님, 패턴 그대로) => 종료

### 예제 분석

- 2
  - 변의 길이 : 5, 3, 3 
- 3
  - 변의 길이 : 13, 7, 7 (4)
- 4
  - 변의 길이 : 29, 15, 15 (8)
- 5 (예측)
  - 변의 길이 : 61, 31, 31 (16)
- N (일반화)
  - 변의 길이 :  2^(N+1) - 3    , 2^N-1, 2^N-1   ( 2^(N-1) )
